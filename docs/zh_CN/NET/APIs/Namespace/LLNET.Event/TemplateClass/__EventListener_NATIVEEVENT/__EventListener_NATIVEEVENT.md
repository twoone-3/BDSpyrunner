# 🔖 __EventListener\<NATIVEEVENT\> 类

## 定义

<br>

>作为 [LLNET.Event](../../LLNET.Event.md) 命名空间中所有**事件监听器**的模板基类。

<br>

C++
```cpp
template<typename NATIVEEVENT>
public ref class __EventListener
```

<br>

## 构造函数

- 
    |||
    |-|-|
    |`__EventListener(Int32)`|使用指定的事件标识初始化对应 `REFEVENT` 事件监听器的新实例。|

<br>

## 方法

- 
    |||
    |-|-|
    |[Remove()]|移除当前事件的监听。|
